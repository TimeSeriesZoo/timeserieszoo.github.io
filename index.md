# Overview

This website hosts visualizers and tutorials for exploring signal processing.

Overall these materials are designed to build up to using signal processing in the context of analyzing neuro-electrophysiological recordings, especially the tutorial. The general materials, including the visualizers may also be useful beyond this use case.

## Source

Most of the materials
The code to make the animations, and copies of the visualizations are available on the
[Github Organization](https://github.com/TimeSeriesZoo/).

Animations are created with the use of the
[NeuroDSP](https://neurodsp-tools.github.io/) module,
an open-source Python module for digital signal processing for neural time series.
